# IGDB Platform IDs

Platform IDs used by the [IGDB API][api] to identify gaming platforms.
Regional variants of the same hardware have separate IDs (e.g. NES vs
Famicom), so configs should include all relevant IDs for best cover art
matching.

[api]: https://api-docs.igdb.com/#platform
